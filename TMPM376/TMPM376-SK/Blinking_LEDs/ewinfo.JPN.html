Example description

  
Blinking LEDs

DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for Toshiba TMPM376 family microcontrollers. It is developed
  for IAR-TMPM376-SK board.

  It uses the WDT to blink the leds on port C.

COMPATIBILITY
=============
  The Leds example project is compatible with IAR-TMPM376-SK board.
  By default the project is configured to use the J-Link JTAG interface.

CONFIGURATION
=============
  Jumpers:
    JP19 - filled
    JP20 - filled
    JP21 - filled
   non isolated JTAG
    JP24 - Filled
   isolated JTAG
    JP24 - no filled
    JP14 - filled
    JP15 - filled
    JP16 - filled

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
