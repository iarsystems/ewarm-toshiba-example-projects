DESCRIPTION
===========
   This template project shows how to use the IAR Embedded Workbench for
   ARM to develop code for Toshiba TC35680FSG microcontroller.
   It demonstrates how to initialize the application and set a breakpoint.

COMPATIBILITY
=============
   By default the project is configured to use the I-Jet/JTAGjet 
   debugger. The example is configured for the TC35680FSG device.

CONFIGURATION
=============
    nothing specific

GETTING STARTED
===============

   1) Start the IAR Embedded Workbench for ARM.

   2) Select Help->Information Center->Example Projects
      Then Donwload and Select or just Select Toshiba, 
    depending on that if you have already downloaded Toshiba examples.
    Then navigate to TC3568x->TC35680FSG and open "TC35680FSG_Template"

   3) "Make" the program.
