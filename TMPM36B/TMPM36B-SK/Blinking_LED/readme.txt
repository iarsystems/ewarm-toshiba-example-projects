DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for
  ARM to develop code for Toshiba TMPM36B family of microcontrollers.

  This example is developed for IAR-TMPM36B-SK Board.

  It uses the WDT to toggle LED on port PF0.

COMPATIBILITY
=============
  By default the project is configured to use the J-Link JTAG interface.

CONFIGURATION
=============
  nothing specific

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
