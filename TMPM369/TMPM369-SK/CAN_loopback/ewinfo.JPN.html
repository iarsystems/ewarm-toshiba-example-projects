Example description

  
CAN Driver and Sample program for TMPM369 device

DESCRIPTION
===========
  This document describes the outline of CAN Driver and Sample program for 
  Toshiba TMPM369 device.

  This program will send the Josystick status data on CAN Bus.
  CAN receiver program will read the User SW button status and reflect
  the status on the LED port.

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
  3) Upper 4 leds will glow continuously.
  4) Move the Josystick in any direction. LEDs will reflect the button status in runtime.
