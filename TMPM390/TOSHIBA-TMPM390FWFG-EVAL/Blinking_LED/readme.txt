DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM to
  develop code for Toshiba TMPM390 family microcontrollers. It is developed for
  Toshiba TMPM390FWFG Eval Board.

   It uses the WDT to toggle the port PJ0.

COMPATIBILITY
=============
  The LED example project is compatible with Toshiba TMPM390FWFG Eval Board.
  By default the project is configured to use the J-Link SWD interface.

CONFIGURATION
=============
  Please set correct SWO CPU clock (9MHz) in J-Link->SWO Settings...
  for correct operation of SWO features

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
