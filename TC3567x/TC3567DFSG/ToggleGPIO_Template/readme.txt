DESCRIPTION
===========
   This template project shows how to use the IAR Embedded Workbench for
   ARM to develop code for Toshiba TC3567DFSG microcontroller.
   It demonstrates a simple use of GPIO module and also shows how to create
   an entry point to the user application.

COMPATIBILITY
=============
   By default the project is configured to use the I-Jet/JTAGjet 
   interface. The example is configured for the TC3567DFSG device.

CONFIGURATION
=============
    nothing specific

GETTING STARTED
===============

   1) Start the IAR Embedded Workbench for ARM.

   2) Select Help->Information Center->Example Projects
      Then Donwload and Select or just Select Toshiba, 
    depending on that if you have already downloaded Toshiba examples.
    Then navigate to TC3567x->TC3567DFSG and open "GPIO Template"

   3) "Make" the program.
