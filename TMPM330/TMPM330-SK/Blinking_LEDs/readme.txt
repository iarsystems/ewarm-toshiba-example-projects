DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for Toshiba TMPM330 family microcontrollers. It is developed
  for IAR TMPM330-SK board.
   It uses the WDT to toggle the port G.

COMPATIBILITY
=============
   The Leds example project is compatible with IAR TMPM330-SK
  board. By default the project is configured to use the J-Link JTAG
  interface.

CONFIGURATION
=============
  Debug iRam - The Progam is loaded to internal RAM memory.
  Debug iFlash - The Progam is loaded to internal flash memory.

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
