DESCRIPTION
===========
  This example project shows how to use the IAR Embedded Workbench for
  ARM to develop code for Toshiba TMPM461 family microcontrollers.
  This example is developed for IAR-TMPM461-SK Board.

  It uses the WDT to display LED port on PD0 pin.

COMPATIBILITY
=============
  By default the project is configured to use the CMSIS-DAP interface.

CONFIGURATION
=============
  nothing specific

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
