DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for
  ARM to develop code for Toshiba TMPM341 family microcontrollers.
  This sample is developed for IAR-TMPM341-SK Board.
   It uses the WDT to toggle the port PE2.
   For properly operation of SWO printf set CPU clock frequency to
  10MHz (internal OSC)

COMPATIBILITY
=============
   The LED example project is compatible with IAR-TMPM341-SK
  board. By default the project is configured to use the J-Link SWD
  interface.

CONFIGURATION
=============
   For properly operation of SWO printf set CPU clock frequency to
  10MHz (internal OSC)

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
