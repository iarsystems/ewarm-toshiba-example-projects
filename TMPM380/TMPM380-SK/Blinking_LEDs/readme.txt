DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for Toshiba TMPM380 family of microcontrollers. It is
  developed for IAR-TMPM380-SK board.

   This example shows basic use of System Timer, interrupts and I/O ports.

COMPATIBILITY
=============
   The Blinking_LEDs example project is compatible with IAR-TMPM380-SK
  board.
   By default the project is configured to use the J-Link JTAG interface.

CONFIGURATION
=============
  Jumpers:
    JP7 - Power source select
    SW3 - 2 (boot from internal flash)

GETTING STARTED
===============
  1) Build and download the example.
  2) Run the program.
