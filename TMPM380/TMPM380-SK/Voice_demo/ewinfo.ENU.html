Example description

  
Voice Demo

DESCRIPTION
===========
   This example project shows how to use the IAR Embedded Workbench for ARM
  to develop code for Toshiba TMPM380 family microcontrollers. It is developed
  for IAR-TMPM380-SK board.
  It repeats a short voice message after 10 seconds pause.
  TSW1  position 3 - playback is running
        position 1 - playback pause
  TSW3  position 3 - enable LEDs indication
        position 1 - disable LEDs indication

COMPATIBILITY
=============
  The Voice Demo project is compatible with IAR-TMPM380-SK board.
  By default the project is configured to use the J-Link JTAG interface.

CONFIGURATION
=============
  Jumpers:
    JP7 - Power source select
    SW3 - A1/B1 (boot from internal flash)

GETTING STARTED
===============
  1) Power the board and connect the J-Link.
  2) Build and download the example.
  3) Run the program.

  NOTE:
  Do not set "Verify download" option in C-SPY debugger when you use
  SPIFlash device. The flashloader program will verify the data internally
  at the time of writting the data to SPIFlash device.
